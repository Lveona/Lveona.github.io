mac部署Node.js环境安装Angular CLI

安装Angular CLI，需要先安装Node.js环境。Angular官网提示node需要8.x，npm 5.x以上的版本。

之前搭建hexo环境也需要node环境，使用homebrew安装的node环境，版本较低，所以需要卸载现有版本重新安装。

删除node和npm

npm config get prefix
cd [刚才得到的路径]
cd lib && rm -rf node_modules
cd /usr/local/lib && sudo rm -rf node*
cd /usr/local/include & sudo rm -rf node*
brew uninstall node (如果你是通过brew安装的，才需要执行此步骤)
cd /usr/local/bin
sudo rm -rf npm
sudo rm -rf node
sudo rm -rf /usr/local/share/man/man1/node.1
sudo rm -rf /usr/local/lib/dtrace/node.d
sudo rm -rf ~/.npm

homebrew安装nvm

安装管理node版本工具有nvm、n，这里选择nvm。使用homebrew
进行安装。后配置环境变量，这里使用的是zsh，在.zshrc这个配置文件中配置，如果是bash，修改.bash_profile文件。

homebrew install -g nvm

vi ~/.zshrc , 在最后一行加上
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"

重新启动配置
source ~/.zshrc

安装node

查看可以安装node版本

nvm ls-romte

安装指定node版本,这里使用淘宝镜像

nvm install node v8.10.0

查看node npm版本

node -v
npm -v

安装Angular CLI

安装@angular/cli

npm install -g @angular/cli

查询安装的Node路径

npm get prefix

把刚才的路径配到你的终端配置文件中.zshrc或.bashrc

alias ng="/刚才你得到的路径/lib/node_modules/@angular/cli/bin/ng"

使用ng命令查看

ng -v


如此angularcli安装完成
